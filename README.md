# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/abhayrajput2005/portfolio.git
```

## Navigate to Project

```bash
cd portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

---

# 📦 Build for Production

```bash
npm run build
```
